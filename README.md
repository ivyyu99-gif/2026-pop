# 📊 지역별 연령별 인구구조 & 전국 유사 지역 분석

행정안전부 주민등록 **연령별 인구현황(월간)** CSV 데이터를 바탕으로,
- 선택한 지역(시/도, 시/군/구, 읍/면/동 모두 포함)의 연령별 인구구조를 꺾은선 그래프로 보여주고
- 그 지역과 **전국에서 인구구조가 가장 비슷한 지역 Top 5**를 찾아 비교해주는 Streamlit 웹앱입니다.

Streamlit Cloud 배포를 기본으로 설계되었습니다.

---

## 주요 기능

### 1️⃣ 선택 지역 인구구조
- 사이드바에서 지역을 검색/선택 (여러 지역 동시 비교 가능)
- 성별(전체 / 남자 / 여자 / 남·여 비교) 선택
- 연령(0~100세, 100세는 "100세 이상") 기준 인구수 꺾은선 그래프 (Plotly)
- 선택 지역 총인구수 요약 테이블

### 2️⃣ 전국 유사 지역 Top 5
- 기준 지역 1곳을 선택하면, 전국 지역 중 **연령별 인구 비율(%)의 유클리드 거리**가 가장 가까운 5개 지역을 찾아줍니다.
- 인구 **규모**가 아니라 인구구조의 **모양**(연령대별 비중)을 비교합니다. 그래야 서울시처럼 인구가 아주 많은 지역과 작은 읍·면·동도 구조적으로 공정하게 비교할 수 있습니다.
- "동일 행정구역 단위끼리만 비교" 옵션(기본 ON): 시/도는 시/도끼리, 시/군/구는 시/군/구끼리, 읍/면/동은 읍/면/동끼리만 비교합니다. 해제하면 단위를 섞어서 비교합니다.
- Top 5 지역의 거리(점수)·총인구수 테이블과, 기준 지역(굵은 실선) + Top 5(점선)를 겹친 Plotly 비교 그래프를 제공합니다.

## 데이터 출처 & 전처리 유의사항

- 데이터: 행정안전부 주민등록 인구통계 사이트에서 내려받은 `연령별인구현황_월간` CSV
- 인코딩: 원본이 보통 `CP949`(EUC-KR)라 앱에서 `cp949 → euc-kr → utf-8` 순서로 자동 판별합니다.
- 숫자 열에 천 단위 구분 콤마(`,`)가 포함된 문자열로 되어 있어, 그대로 `pandas.describe()`를 돌리면 콤마 없는 일부 열만 숫자로 인식되는 문제가 있었습니다 → 콤마 제거 후 숫자형으로 변환하도록 처리했습니다.
- 행정구역명에 괄호로 붙은 행정코드(예: `(1100000000)`)는 **지역명 유일성 보장을 위해 그대로 유지**합니다. (코드를 제거하면 "세종특별자치시"처럼 시/도 레벨과 시 레벨 이름이 겹치는 경우가 있어 오류가 발생했습니다.)
- 행정구역 단위(시도/시군구/읍면동)는 지역명의 공백 구분 토큰 개수로 자동 분류합니다.

## 파일 구조 (저장소 루트에 모두 같은 위치)

```
├── main.py               # Streamlit 앱 본체
├── requirements.txt      # 의존 패키지
├── 202606_202606_연령별인구현황_월간.csv   # 데이터 파일 (원본 파일명 그대로)
└── README.md
```

> ⚠️ `pages/` 폴더를 만들어 멀티페이지 구조로 쓰는 경우에도, `main.py`와 CSV는 **저장소 루트**에 두면 됩니다. 앱 코드가 (1) 실행 중인 작업 폴더 (2) 스크립트 파일 폴더 (3) 스크립트 파일 폴더의 상위 폴더, 순서로 데이터 파일을 자동으로 찾습니다.

## 데이터 파일을 못 찾는다는 오류가 뜰 때

앱 화면의 "🔍 진단 정보"를 펼치면, 코드가 실제로 확인한 폴더들과 그 안의 파일 목록을 보여줍니다. 아래 순서로 확인해 보세요.

1. CSV 파일이 저장소(GitHub)에 실제로 커밋되어 있는지
2. 파일명이 정확히 일치하는지 (공백, 오타 여부)
3. 한글 파일명의 유니코드 정규화 차이 — macOS(NFD)와 GitHub/Linux(NFC)에서 같은 글자라도 바이트가 달라 `os.path.exists()`가 실패할 수 있습니다. 앱이 이를 자동으로 보정하지만, 그래도 안 될 경우 파일을 한 번 다시 업로드/커밋해 보세요.

## 유사 지역 계산 방법 (참고)

1. 지역별로 연령(0~100세) 인구수를 해당 지역 총인구수로 나눠 **비율 벡터**로 정규화
2. 기준 지역과 비교 대상 지역 간 비율 벡터의 **유클리드 거리** 계산
3. 거리가 작은 순서대로 Top 5 선정 (거리 = 0에 가까울수록 인구구조가 유사)

필요 시 유클리드 거리 대신 코사인 유사도, 상관계수 등 다른 지표로 바꿀 수 있습니다.
